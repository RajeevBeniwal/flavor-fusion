🚀 Publish Data to Website

All your changes (menu items, categories, restaurant info) are stored in your browser. To make them visible to all customers on any device, you must publish by uploading data.json to GitHub.

📋 How to Publish (repeat every time you make changes)

  1. Click "📥 Download data.json" below
  2. Open your GitHub repository in browser
  3. Find data.json → click it → click ✏️ Edit
  4. Select all → Delete → Paste content from downloaded file
  5. Click "Commit changes" — done in 1–2 minutes ✅
💡 Faster way: Drag the downloaded data.json file onto your GitHub repo page → it will prompt to commit → done!
